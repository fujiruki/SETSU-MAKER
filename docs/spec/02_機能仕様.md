# 02. 機能仕様

## 機能一覧

| 機能 | 概要 |
|---|---|
| ノート編集 | 工程CRUD・並び替え・ハイライト・写真・ヒント |
| ノート閲覧 | ヒント切替・ライトボックス・お気に入り |
| ダッシュボード | 最近のノート・お気に入り・全件一覧・検索・タグ |
| カテゴリ管理 | ツリー構造・CRUD・エクスプローラー表示 |
| メディア管理 | 画像/動画アップロード・EXIF・トリミング・アノテーション・サムネイル |
| アプリ設定 | 動画出力形式・最大長さ・サムネイルサイズ |

---

## ノート編集

### 基本操作

- タイトル編集
- カテゴリ割り当て
- タグの追加・削除（入力＋Enterで確定、サジェスト付き）

### 工程（Step）操作

- **追加:** 末尾に新しい工程を追加
- **挿入:** 工程間の境界線中央にある＋ボタンをクリックして、任意の位置に工程を挿入
- **削除:** 工程の削除
- **並び替え:** 工程左側のグリップアイコンをドラッグして順序変更（@dnd-kit使用）。表示領域の上端・下端までドラッグするとスクロールが始まる
- **編集:** 工程タイトル・説明文の編集

### ハイライトブロック

工程の説明文内に挿入できる強調ブロック。3種類:

| タイプ | 表示 | 用途 |
|---|---|---|
| `warning` | ！注意 | 安全上の注意点、失敗しやすいポイント |
| `point` | Point! | コツ、効率化のヒント |
| `note` | メモ | 補足情報 |

各ブロックはCRUD可能。

### ヒント欄

- 各工程の右側にテキストエリアを配置
- 閲覧画面でヒント編集モードに切り替えて編集可能
- 作業者が気づいたコツや注意点を自由に書き込む欄

### 写真の削除

- 編集モードで各画像の右上に×ボタンを表示
- クリックで該当写真をステップから削除

### Undo（元に戻す）

- ヘッダーに「元に戻す」ボタンを配置
- 編集操作（テキスト変更・写真追加/削除・工程追加/削除/並び替えなど）を1つずつ巻き戻す
- 間違いを恐れず作業できるようにする目的

### 保存

- 保存ボタンクリックで保存
- 保存成功時に「保存されました HH:MM」と小さく表示
- 保存ボタンの横に閲覧モードへ戻るボタンを配置

## ノート閲覧

### 表示モード

- **通常モード:** 工程内容を表示（読み取り専用）
- **ヒント編集モード:** ヒント欄のみ編集可能

ヘッダーに「編集」「ヒント編集」ボタンを配置し、モード切替。

### 画像表示

- 画像の表示幅はノート内で統一
- トリミング範囲がある場合、その範囲のみを表示幅に合わせて表示
- 高さはトリミング範囲（または元画像）のアスペクト比に従う
- クリックでライトボックスを開き全体像を表示

### お気に入り

- ノートにお気に入りフラグを設定可能
- ダッシュボードのお気に入りセクションに表示

|---|---|---|

- デスクトップ: 左サイドバーにナビゲーション

- カテゴリエクスプローラーへのリンクを含む

### 検索

- ノートタイトルによるテキスト検索

## カテゴリ管理

### ツリー構造

- カテゴリは親子関係を持つ（`parentId`による階層構造）
- 各カテゴリに`order`フィールドで並び順を管理

### デフォルトカテゴリ「未分類」

- ID固定値: `uncategorized`
- バックエンド初回起動時に自動作成（存在しなければ）
- **削除不可・名前変更不可**
- カテゴリツリーの**一番下**に常に表示
- 新規ノート作成時のデフォルトカテゴリ

### カテゴリエクスプローラー

- **左ペイン:** ツリー構造でカテゴリを表示。ノードのクリックで展開・折りたたみ
- **右ペイン:** 選択中カテゴリのサブカテゴリとノートを表示
- **ノート作成ボタン:** 特定カテゴリを閲覧中に押すと、そのカテゴリを初期設定した状態でノート編集画面を開く

### CRUD

- カテゴリの作成（名前・親カテゴリ指定）
- カテゴリの名前変更（「未分類」を除く）
- カテゴリの削除（「未分類」を除く）
- パンくずリスト表示

## 画像管理

### アイキャッチ画像

- ノート編集画面でアイキャッチ画像を選択可能
- 「アイキャッチ画像を選ぶ」ボタンでピッカーモーダルを表示
- ピッカーモーダル: ノート内の全写真をグリッド表示し、1つを選択
- **未選択の場合**: ノート内の最後の写真を自動的にアイキャッチとして使用
- ダッシュボード・カテゴリエクスプローラーのノートカードに表示

### 一括アップロードと未割り当て写真プール

- 複数写真を一括選択してアップロード可能
- アップロードした写真はEXIF撮影日順にソートされ、**未割り当て写真プール**に格納
- 未割り当てプールは全工程の下に一覧表示
- 未割り当て写真の操作:
  - 写真の「+」ボタン → 新しい工程を作成し、その写真を割り当て
  - 右クリック → コンテキストメニューから既存の工程に割り当て
- 未割り当て写真は保存時にサーバーにアップロードされ、永続化される

### アップロード方式

写真追加メニューから3方式を選択:

| 方式 | 説明 |
|---|---|
| 写真を撮る | `capture="environment"` でカメラ起動（タブレット・スマホ向け） |
| アップロード | ファイル選択ダイアログ（複数選択可） |
| クリップボードから追加 | Clipboard APIで画像を取得。画像がない場合はアラート表示 |

メニュー外をクリックすると閉じる。

### 画像処理

- EXIF情報から撮影日時を取得し、ファイル名に使用（`20260319_143000_a3f2b1.jpg` 形式）
- EXIFがない場合はアップロード日時を使用
- ノートごとにフォルダ分けして保存（`uploads/{noteId}/`）

### 画像編集（ライトボックス）

`@fujiruki/react-image-editor-lightbox` を使用:

- **矢印追加:** 画像上に矢印を描画
- **テキスト追加:** 画像上にテキストを配置
- **トリミング:** ハンドルをドラッグしてトリミング範囲を指定。実際の切り抜きは行わず、表示範囲として保持
- 閉じるボタン: 右上、画面端から50px離した位置に配置。通常の2倍サイズで四角形に囲んで表示

### 画像編集の表示制御

- 画像編集を開いた時、画像は**画面の90%に収まるサイズ**に縮小して表示（オリジナルサイズで画面外にはみ出さない）
- **ピンチイン・ピンチアウト**で画像を拡大・縮小して確認可能（表示倍率の変更のみ、トリミングではない）

### 画像の並び順

- アップロード直後は撮影日時順（EXIF `DateTimeOriginal`）
- 各工程内で `order` フィールドによる並び順管理

### サムネイル生成

- 画像アップロード時にフロントエンドでCanvas APIにより長辺480pxのサムネイルを自動生成
- サムネイルはサーバーの `uploads/{noteId}/thumb/` に保存
- ダッシュボードのアイキャッチ、StepCardのサムネイル表示ではサムネイルURLを優先使用
- サムネイルサイズはアプリ設定で変更可能（320/480/640px）

### 既存画像の一括サムネイル生成

- サムネイル導入前にアップロードされた画像にはサムネイルが存在しない
- サーバー上でPHP GDを使い、`uploads/*/` 内の画像から `uploads/*/thumb/` にサムネイルを一括生成するスクリプトを提供
- 実行: SSH経由またはブラウザからの管理用エンドポイント
- 生成後、ノートのJSONデータ内の各photoに `thumbnailUrl` を自動付与

## 動画対応

### 概要

施工手順の「動き」を伝えるために、3秒〜1分程度の短い動画をステップに添付可能。

### 対応形式

MP4, WebM, MOV のアップロードに対応。

### 動画トリミングUI

- 動画ファイル選択時にiPhone風トリミングUIが自動で開く
- フィルムストリップ（サムネイル列）の両端を黄色いハンドルでドラッグして範囲選択
- 白いプレイヘッドで再生位置を確認
- 「スキップ」で元の動画をそのままアップロード
- 「トリミング」でFFmpeg.wasmを使いブラウザ内で動画をカット処理
- 進捗バーでトリミング処理の進行状況を表示

### 表示

- StepCard内では動画サムネイル + 再生アイコン（Play）で表示
- 閲覧画面でクリックするとライトボックスで動画再生（controls付き、autoplay）

### データモデル

Photo型に `mediaType: 'image' | 'video'` と `duration: number` フィールドを追加。

## アプリ設定

### ルート

`/settings`

### 設定項目

| 設定 | 選択肢 | デフォルト |
|---|---|---|
| 動画出力形式 | MP4 / GIF | MP4 |
| 動画の最大長さ | 15秒 / 30秒 / 60秒 | 60秒 |
| 画像サムネイルサイズ | 小(320px) / 中(480px) / 大(640px) | 中(480px) |

### 保存先

LocalStorage（キー: `sm_settings`）

### UI

- ダッシュボードのサイドバーに「設定」リンクを配置
- セクション別にグループ化されたラジオボタン選択式

---

## テキストエリア自動拡張

- 編集モードのtextarea（説明・ヒント）は内容に合わせて高さが自動拡張する
- scrollHeightベースで計算、上限 max-height: 50vh
- 上限超過後はtextarea内スクロールにフォールバック
- 閲覧モードでは `<p>` タグで全文表示（textareaを使わない）
- 初期表示時にも内容量に合わせた高さで表示

---

## 動画サムネイル

- 動画アップロード時にCanvasで先頭フレームをキャプチャ → JPEG化 → サムネイルとしてアップロード
- StepCard内の動画表示: サムネイル画像 + 暗いオーバーレイ(30%) + 再生ボタン（従来の真っ黒背景を廃止）

---

## 保存プログレスバー

- ヘッダー直下に2pxの薄いプログレスバー（sticky固定）
- 青色で進行表示、完了後に緑色に変わりフェードアウト
- アップロード中のbeforeunload警告を追加
- 個々のファイルアップロード進捗を積算して全体進捗を表示
