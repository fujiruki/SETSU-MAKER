# 対象ユーザー
## 1. 製造業で社員に作業工程を教えるためにマニュアルをつくる人
### 目的
- 社員が理解しやすいマニュアルを作る。できるだけ時間をかけずに作る。
- 後日の自分が忘れても思い出せるようにする。
## 2. 製造業で社員として、作業が正しいか確認するためにたまにマニュアルを読みながら作業をする人
目的：作業を正しく完了することが目的である。
- 注意点を読み、自分や仲間が書いたヒントを元に間違いやすいところを確認する。
- マニュアルの改良も行う。
## 3. 製造業で社員として、初めてその作業をするために、まずマニュアルを読む人
目的：全体像とコツの部分を掴み、教えてもらうための予習をする。
